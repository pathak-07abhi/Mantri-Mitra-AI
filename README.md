**Mantri Mitra AI** is an AI-powered digital assistant designed to support **Indian MLAs, MPs, and government administrators** in managing constituency issues, official communication, and public data insights.

The platform acts as a **smart governance companion**, helping leaders process information faster, generate policy communication, and monitor citizen concerns efficiently.

---

# 🏛 Vision

Public leaders deal with thousands of documents, citizen complaints, and administrative decisions every day.

**Mantri Mitra AI** simplifies governance by providing:

- Intelligent document analysis
- AI-generated speeches and official responses
- Issue tracking for citizen grievances
- Meeting intelligence and summaries
- Data insights for constituency management

The goal is to enable **faster, smarter, and more transparent governance.**

---

# 🚀 Key Features

## 📊 Constituency Issue Tracker
Track and analyze public complaints and generate **AI-assisted resolution strategies**.

## 📄 Document Intelligence
Upload official documents and instantly:
- Summarize them
- Translate them
- Ask questions about them
- Generate briefing notes

## 🗣 Speech Generator
Create **official speeches, announcements, and public statements** in multiple Indian languages.

## 📝 Meeting Intelligence
Automatically generate:
- Meeting agendas
- Structured summaries
- Action points

## 📈 Governance Dashboard
Visual insights for:
- Public sentiment
- Issue trends
- Administrative priorities

## 🔐 Secure Authentication
Uses **SHA-256 hashing via Web Crypto API** for secure login handling.

## 📱 Mobile Friendly
Works smoothly across:
- Desktop
- Tablets
- Smartphones

## 🌙 Dark Mode
Built-in theme toggle for modern interface.
